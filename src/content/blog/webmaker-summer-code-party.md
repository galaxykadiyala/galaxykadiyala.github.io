Mozilla announced its new [Webmaker](https://webmaker.org/) initiative and it's something special.

**Mozilla Webmaker** is a project to help people make something creative, learn new things, and play using open web technologies.

> *The goal: help millions of people move from **using** the web to **making** the web. With new tools to use, projects to create, and events to join, we want to help the world increase their understanding of the web and take greater control of their online lives.*

## Summer Code Party

The Summer Code Party kicks off on June 23rd — an open invitation to meet up with friends, youth, or anyone, make cool stuff, and learn how the code behind the web works.

The tools available are great: **Hackasaurus** for remixing websites with X-ray Goggles, **Popcorn** for remixing video, and **Thimble** — an awesome new tool that's basically a web page editor combined with projects that teach HTML and CSS basics.

Hosting an event is simple. A Kitchen Table event needs just a couple of enthusiastic people, 20 minutes, and a table.

Partners include Tumblr, Creative Commons, and others. The open web has good friends.

Go [find or host an event](https://events.webmaker.org/) near you. Start hacking on the web!
