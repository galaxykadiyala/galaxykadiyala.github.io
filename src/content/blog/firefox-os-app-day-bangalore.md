This short video was created entirely with [Mozilla Popcorn Maker](https://popcorn.webmaker.org/) to document Firefox OS App Days Bangalore.

It was one of my first experiments with Popcorn Maker — a tool that lets you remix and annotate video entirely in the browser, no software required. The fact that this event recap was itself made with a Mozilla open web tool felt very on-brand.

Firefox OS App Days were happening all around India at this time. Developers were getting their first hands-on experience building apps for Firefox OS using just HTML, CSS, and JavaScript — proving that the open web could power a mobile operating system.

Exciting times.
