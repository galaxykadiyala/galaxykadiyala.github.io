One of our Mozilla Student Reps, Vijesh from NMIMS Bangalore, did an install fest at his college and updated all the machines to the latest Firefox browser!

This is exactly the kind of grassroots action the Firefox Student Ambassador program is designed to enable. No big budget, no corporate backing — just a student who cares about the open web, walking into a computer lab and making it better.

Install fests like this were one of the most direct ways the Mozilla community made an impact. Every machine running Firefox instead of IE was a small win for the open web.
